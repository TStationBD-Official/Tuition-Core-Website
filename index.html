Powerful Features

📝

Advanced Exam System

Create exams with MCQ & CQ, auto grading, shuffled questions, student reviews, and deterministic randomization.

👥

Student Management

Manage student rosters with subscription limits, track attendance, monitor progress, and send assignments.

📅

Class Scheduling

Organize class routines with calendar integration, automatic attendance, and real-time updates.

🔔

Multi-Device Notifications

Real-time notifications via Firebase across iOS, Android, Web, Windows, macOS, and Linux.

💳

Subscription Management

Complete billing system with 5 tiers, monthly/yearly plans, add-ons, and payment tracking.

☁️

Google Drive Integration

Secure file storage with MD5 deduplication, automatic syncing, and persistent access.

Guardian Monitoring

Fee Management

  • Post announcements
